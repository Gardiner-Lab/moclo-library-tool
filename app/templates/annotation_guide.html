GenBank Annotation Best Practices

Guidelines for annotating parts with introns and other features

Annotating Introns in Coding Sequences

When your coding sequence contains introns, proper annotation is essential for accurate translation analysis. The tool will automatically detect and process introns to show both genomic and spliced sequences.

Intron Annotation Requirements

Important: Intron annotations must include the splice sites (GT at the 5' donor site and AG at the 3' acceptor site) as part of the annotated region.

Visual Example: Gene with Intron

Genomic DNA Sequence:
ATGAAGCTGGTAAGT...TTTCAGTACGGATAA
● Start codon (ATG)   ● Exons   ● Intron   ● Stop codon (TAA)
Annotation Structure:
Position 1-300:
CDS (Complete Gene)
Position 50-150:
Intron
Splice Site Detail:
5' Donor Site (position 50):
...AAGCTGGTAAGT...
                    ↑ Intron starts here (includes GT)
3' Acceptor Site (position 150):
...TTTCAGTACGGA...
                  ↑ Intron ends here (includes AG)
After Splicing (mRNA):
ATGAAGCTGTACGGATAA
Intron removed, exons joined seamlessly

GenBank Format Example

FEATURES             Location/Qualifiers
     CDS             1..300
                     /label="MyGene"
                     /translation="MKKL..."
     intron          50..150
                     /label="intron1"
                     /note="First intron - includes GT...AG"
✓ Correct: The intron annotation spans from position 50 to 150, which includes:
  • The GT at positions 50-51 (donor site)
  • The internal intron sequence
  • The AG at positions 149-150 (acceptor site)
✗ Incorrect: Do NOT annotate from position 52 to 148 (excluding splice sites). The tool expects the complete intron including GT and AG.

Key Points

  • Feature Type: Use "intron" as the feature type
  • Coordinates: Include the complete intron sequence including GT...AG splice sites
  • Label: Provide a descriptive label (e.g., "intron1", "cdpk3_intron")
  • Multiple Introns: Each intron should be a separate feature annotation

Splice Site Context

The tool will display splice site context for each intron:

  • Donor Site: Shows 8bp context around the 5' GT splice site
  • Acceptor Site: Shows 8bp context around the 3' AG splice site
  • Consensus Sequences: Typical patterns are MAG|GTRAGT (donor) and YAG|R (acceptor)
💡 Tip: The tool automatically extracts intron annotations from GenBank files. You don't need to manually specify intron positions when uploading properly annotated files.

Translation Analysis with Introns

When a cassette contains coding sequences with introns, the tool provides comprehensive translation analysis:

What You'll See

  1. Genomic Sequence: The complete DNA sequence including introns
  2. Spliced mRNA: The sequence after intron removal
  3. Protein Translation: Amino acid sequence from the spliced mRNA
  4. Splice Site Table: Context sequences for each donor and acceptor site

Intron Detection Methods

The tool detects introns through:

  • GenBank Annotations: Intron features in uploaded GenBank files (recommended)
  • Part Names: Parts with "intron" in the name (case-insensitive, word boundary match)
  • Part Types: Parts classified as "NonCodingIntron"
⚠️ Note: The tool does NOT automatically predict intron locations. All introns must be explicitly annotated in the GenBank file or specified through part classification.

General Best Practices

File Preparation

  • Use standard GenBank format (.gb or .gbk files)
  • Include descriptive labels for all features
  • Verify sequence accuracy before upload
  • Check that coordinates are correct (1-indexed)

Naming Conventions

  • Use clear, descriptive names for parts and features
  • Include organism or source information when relevant
  • Avoid special characters that might cause parsing issues
  • Be consistent with naming across your library

Quality Control

  • Review translation analysis after upload to verify intron processing
  • Check splice site sequences match consensus patterns
  • Verify that spliced sequence produces expected protein
  • Test assembly compatibility before large-scale use
